# Legacy Onboarding Steps

This folder contains backup copies of the previous onboarding step components.
These files are preserved for reference but are not actively used in the application.

The new onboarding flow uses the following steps:
- `ModeSelectionStep.tsx` - Choose between guided and manual setup
- `ContextIntentStep.tsx` - Gather context and intent information
- `VisualDirectionStep.tsx` - Collect visual direction preferences
- `TearSheetIntentStep.tsx` - Define tear sheet purpose
- `GeneratingStep.tsx` - Generate moodboard and project

The legacy steps (`.tsx.bak` files) were:
- `AssignmentStep.tsx` - Project naming and directive
- `BudgetStep.tsx` - Budget policy configuration
- `CloseStep.tsx` - Legacy completion/generation step
- `DesksStep.tsx` - Desk selection
- `DiscoveryStep.tsx` - User discovery phase
- `EditorsReadStep.tsx` - Editors read configuration
- `FirstGenStep.tsx` - First generation step
- `ToneCheckStep.tsx` - Tone and voice settings
