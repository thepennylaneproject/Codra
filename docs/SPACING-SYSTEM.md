# Spacing System (8px Rhythm)

Codra spacing uses a locked 4px base scale with 8px rhythm. Only the
following steps are permitted in layout utilities and component spacing.

## Scale

- 0px: `0`
- 4px: `1`
- 8px: `2`
- 12px: `3`
- 16px: `4`
- 24px: `6`
- 32px: `8`
- 48px: `12`

## Usage Guidance

- 4px: tight spacing within components (icon + label, compact badges)
- 8px: standard spacing between related elements
- 12px: small group separation and dense stacks
- 16px: default section spacing and card padding
- 24px: major section breaks and grouped blocks
- 32px: large layout separation
- 48px: page-level separation and hero sections

## Notes

- Avoid arbitrary spacing values or decimals.
- Prefer consistent spacing inside component families.
